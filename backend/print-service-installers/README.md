# Instaladores del servicio de impresión

Coloca aquí los archivos descargables del **AgoraOps Print Service** (p. ej.
`AgoraOpsPrintService-1.0.0-win-x64.zip`). La pantalla *Gestión de cajas →
Descargar servicio de impresión* lista automáticamente los archivos de esta
carpeta (nombre, tamaño, tipo y fecha) y permite descargarlos.

Para generar el `.exe`: ver `print-service/README.md` (`npm run build:win`),
comprimirlo en ZIP y dejarlo en esta carpeta.

(Este README y los archivos ocultos no se muestran en la lista.)
